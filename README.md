# Local-AI-Chatbot-LLaMA3-LangChain-Streamlit
A local AI chatbot built using LangChain, Ollama (LLaMA3), and Streamlit. This application runs fully offline and demonstrates prompt chaining and output parsing using LangChain core components.

Prequisists:
Install ollama in local environment and install llama3 model.

Create API keys which are required for project and create .env file replace with your key values there.

Download the libraries which are present in requirements.txt file.

Set up the streamlit env.

Run the program with streamlit.
